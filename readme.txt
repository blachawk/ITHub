== Description ==

Dynamic Content Assistance: IT Hub Assistant is a plugin that was created to help speed up the content update process on external servers. Think of it as a helper for dynamically displaying content right away. This requires ACF to be activated. It also requires UnderStrap to be the main theme. At the moment there no option fields to manage. Please check back later for option updates.

= IT Hub Assistance Feature Highlights =

* Automatically ports ACF object data onto existing pages without modifying the templates on external severs.
* Provides search assistance on the search page, based on keywords typed into the search input field.
* Customizes the attachment post type
* Comes with a flexible BootStrap 4 framework for the settings page and front end pages.
